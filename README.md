# CodiceFiscale

## EN

#### A Java GUI application in italian that lets you generate a codice fiscale (fiscal code, unique ID for every Italian citizen).

I'm stil very new to programming (less than a year's experience), and this is my very first thing done with Java Swing, so don't expect much.
I created this just beacuse I wanted to try and learn something about Swing, while still applying it to something that actually has a real use, like a fiscal code generator.

Comments and Javadocs are both in English and Italian.

To learn more about how the fiscal code's structure, check the Wikipedia page: https://it.wikipedia.org/wiki/Codice_fiscale
